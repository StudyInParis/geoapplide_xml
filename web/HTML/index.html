Génèse

Kézako ?

Nous présentons notre projet réalisé dans le cadre de notre M2 TAL à l'INALCO. Notre projet était de travailler sur les données libres, nous avons donc utilisé les sites d'open data data.gouv.fr et le site de données publiques de Paris.

A partir de ces données, nous avons réalisé plusieurs traitements sur les données pour pouvoir les utiliser ensuite. C'est à partir des données récupérées que nous avons décidé de les utiliser pour déterminer le quartier de Paris le plus attractif pour les étudiants.
